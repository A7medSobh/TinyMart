# TinyMart OOP Product Catalog

## Overview
This is a C# implementation of the TinyMart product catalog system. It demonstrates Object-Oriented Programming (OOP) principles including inheritance, polymorphism, encapsulation, and operator overloading.

## Files Included
* `NameType.cs` - Contains structs and enums.
* `Product.cs`, `AudioProduct.cs`, `VideoProduct.cs`, `BookProduct.cs`, `EBook.cs`, `PaperBook.cs` - The product class hierarchy.
* `Cart.cs` - Manages the collection of products and file I/O.
* `Exceptions.cs` - Custom exception handling.
* `Program.cs` - The main test driver.
